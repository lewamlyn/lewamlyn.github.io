Just for fun！

想写这篇文章很久了，现在在各项内容基本配置完成后，终于有时间来写完了。

v 0.0

最开始有建一个博客的想法是从想写些自己想写的东西开始的。人毕竟是社交动物嘛，写的东西总是想与人分享，但真的想要下键盘写些东西的时候才发现，对我而言，却没有一个地方适合发些自己的想法和记录。QQ 嘛，太花哨，而且不是很适合记录长文；微信嘛，加的长辈太多，我又不善与人理辩，于是也只能放弃。而且作为社交软件，这二者加的人太多太杂，总之，多有不便。

一开始的想法是建个微信公众号，我也查了微信公众号的建立流程，感觉也挺麻烦的，还要被麻花腾薅羊毛，于是就放弃了这一想法。写东西的想法也随之搁置了。

后来真正促使我建立的博客的契机，还是坑大的 SRTP 项目。导师要求所有人都建立 gayhub 账号，然后把我们拉进一个项目里，要求以后相关内容的推进都在gayhub上完成。于是，我只好终止了白嫖代码的时代，第一次创建了我的 gayhub 账号。

然而，最终建立博客的决定是摸鱼摸出来的。2020/06/05，即周五上完课后，准备做课程大作业时，习惯性的刷起了 CC98（坑大唯一官方论坛），看见一位论坛老哥的个性签名是他的博客地址，点进去欣赏了之后被“震撼”了，这就是大佬的世界吗？

这么想来，或许，个人博客才是写些东西的最佳选择，想写就写，无论是可能存在的技术文档（虽然可能性不大），还是个人感想和随笔记录，来看的永远是真的想阅读我的文字的。

所以，给我也整一个.jpg

来都来了，反正也是摸鱼，不如摸点有意思的鱼，just for fun！

v 1.0

由于那位大佬的博客隐藏了版权信息，我也不知道他是咋整出来的，只能看到地址中的 github。于是只能用搜索引擎搜索 github 搭建博客的信息。如果我没记错的话，github page 官方推荐的是 jekyll 搭建，于是我又搜索如何用 jekyll 搭建博客，最后摸鱼一直摸到晚上，才最终整了个半成品出来，以目前的眼光来看，只能说勉强能用吧，有了基本功能，但不够个性化，太过朴素（不够 cool ）。

第二天又开始摸鱼，在 CC98 上找了个 Markdown 的语法简单教程，总结（抄袭）了过来，成了第一篇有意义的博客。

摸了两天鱼后，实在摸不动了。随着期末的来临，只能转身投入到知识的海洋中，准备烤柿粥去了。

v 2.0

2020/06/26 下午 2 点进行最后一场考试，虽然 9 点多才起床，为了得到充足的休息时间，我 11 点多就又上了床准备午觉，可是不知是终于要迎来暑假的兴奋，还是终于可以真正完善我的博客的激动，一直到下午一点也没能入眠，最后也只能这样去考试了。

为了不愧对自己最后一场考试的牺牲，考完后夜以继日一直调试、配置，直到差不多满意为止（2020/06/30）。老话确实没说错，兴趣果然是“孩子”最好的老师。

最开始见那位大佬的博客以为要弄很久，准备暑假多花些时间研究学习的，最后其实也还好，感谢所有大佬和 github （开源是程序员最好的美德）。

现在，终于可以享受乐趣了（后续可能仍有修改）。

Just for fun，for more fun!
